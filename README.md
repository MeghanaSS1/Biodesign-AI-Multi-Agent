# Biodesign-AI-Multi-Agent
This project presents a Biodesign AI Multi-Agent System designed to support the early Identify phase of healthcare innovation. The system begins with a clinical observation and converts it into a structured, debatable, and verifiable need portfolio.

The prototype uses a seven-agent structure. The Intake Analyzer first organizes the observation into a case packet. Three domain experts then debate the case: Clinical-Technical, Regulatory-Business, and Human-Patient. Additional agents support option formation, critical review, and final evaluation. The final output is an STB Identify Portfolio containing a solution-free Need Statement, Need Profile, MCDM screening, evidence gaps, and a validation plan.
Core project value
The system does not decide the medical solution for the user. Its value is to help the team see blind spots earlier, test assumptions, and produce a need portfolio that can be reviewed by classmates, mentors, or clinical advisors.


Methodological idea:
The system uses debate as a method of quality control. Better output is defined not only by fluency, but by whether risks, assumptions, evidence gaps, and validation tasks become clearer.

Input	: 	
Clinical observation, care setting, stakeholders, constraints, evidence		

Process : 
Case Packet, expert positions, cross-challenge, evidence gaps, MCDM

Output : 
Need Statement, Need Profile, Need Screening, Validation Plan, Need Criteria

Agent group	Main responsibility
Intake Analyzer :	Structures observation into case packet and debate agenda.
Domain Experts : Clinical-Technical, Regulatory-Business, and Human-Patient experts state positions and challenge each other.
Review and Evaluation :	Critical Review identifies assumptions; Consolidated Evaluator creates final portfolio.

Final takeaway:
The Biodesign AI Multi-Agent System turns clinical observations into a debatable, screenable, and verifiable early-stage need portfolio. The system supports human decision-making by making assumptions, risks, evidence gaps, and validation tasks easier to see.
